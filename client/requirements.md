## Packages
framer-motion | Smooth, calming animations for transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
- The app targets autistic users, so the UI must be calm, predictable, and avoid flashing.
- Use a "calm" color palette (blues, greens, soft grays).
- API endpoints are defined in `@shared/routes`.
- Fonts should be highly legible (Sans-serif).
